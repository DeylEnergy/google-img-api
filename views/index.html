Image Search Abstraction Layer

How to use

In order to grab the request, url should look like this:

https://image-search-abstr-layer.glitch.me/api/imagesearch/cats

To to paginate through results you must add to url ?offset=n, where n is quantity of ommited results n > 1

https://image-search-abstr-layer.glitch.me/api/imagesearch/cats?offset=11

To show the latest queries:

https://image-search-abstr-layer.glitch.me/api/imagesearch/cats?offset=11
